# Sistema de Saque e Depósito em C

Projeto desenvolvido em linguagem C para simular operações bancárias básicas como saque, depósito e consulta de saldo. O objetivo foi praticar lógica de programação e estruturas de controle.

## Funcionalidades

- Depósito de valores
- Saque de valores
- Consulta de saldo
- Menu interativo

## Tecnologias utilizadas

- Linguagem C
- VS Code
- Git/GitHub

## Conceitos praticados

- Estruturas condicionais
- Estruturas de repetição
- Funções
- Lógica de programação

## Como executar

gcc sistema.c -o sistema

./sistema
